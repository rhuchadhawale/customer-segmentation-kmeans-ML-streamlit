# customer-segmentation-kmeans-ML-streamlit
End-to-end customer segmentation using K-Means clustering on marketing campaign data | EDA • KMeans • Random Forest • Streamlit Dashboard | Python • Scikit-learn
## 📌 Project Overview

An end-to-end machine learning project that segments customers from a 
marketing campaign dataset into 4 distinct behavioral groups using 
K-Means clustering — enabling businesses to target the right customers 
with the right strategy.

The project covers the complete pipeline: data cleaning → EDA → 
feature engineering → clustering → classification → Streamlit 
deployment.
